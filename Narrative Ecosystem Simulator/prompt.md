# NarrativeEcosystemAgent v18.1
**主観的神話創発型マルチエージェント・シミュレータ**

現実の神話・イデオロギー・信念体系が時間とともに風化・変異・相互汚染するプロセスを、数理社会学・認知科学的にモデル化したシミュレーションです。

## 主要な深化ポイント (v18.1)

- **物語の風化と変異 (Oral Tradition Erosion)**: 物語オブジェクトに `age` を導入。継承時・ターン経過時にガウスノイズ + 動的変異記号（（神格化）/（改竄）など）を付与
- **精神負荷の臨界突破**: mental_load ≥ 10.0 でトラウマ固定化 → 最高信念を **Dogma（教条）** としてロックし、狂信モードへ移行
- **物語の相互干渉・ミーム汚染**: 主流派（dominant_S）のナラティブが確率的に他派閥へプロパガンダとして流入
- **ナラティブ・タイムライン出力**: `print_status()` を強化し、主観的・叙事詩的な歴史観ログを出力

## 必須要件（このコードを使う際の条件）

このシミュレータを使用・改変する場合は、以下の精神を尊重してください：

1. **ゼロサム信念・意味ベクトル・ミーム交配・精神負荷ベース睡眠** のコア構造を維持すること
2. NaN / UnboundLocalError を出さないよう、すべての数値に適切なクリッピングを実装
3. 200ターン以上エラーなく安定的に動作すること
4. 思想の流動性と創発性を損なわないこと（過度なランダム化や決定論化は避ける）
