HeuristicExplorerAI
概要
HeuristicExplorerAI は「存在論的空洞（Void）」を行動原理として持つ実験的エージェントである。

本モデルの目的は、単なる報酬最大化や恒常性維持ではなく、

理解すると退屈する
退屈すると行動する
行動すると世界が変化する
変化した世界を再び理解する
という循環が成立するかを検証することであった。

モデルの構造
Void（存在論的空洞）は4つの欠損から構成される。

1. 生存欠損（Deprivation）
温度や湿度が許容範囲から外れた際に生じる苦痛。

2. 予測欠損（Uncertainty）
世界モデルの予測誤差から生じる不安。

3. 存在欠損（Novelty Hunger）
世界を理解したことによって発生する退屈。

本モデルの中核概念。

4. 制御欠損（Control Error）
理想状態からのズレや非効率性への飢え。

中核仮説
世界を理解するほど退屈が増大する。

退屈は現状維持への評価を下げ、改善余地の探索を促す。

つまり、

理解 → 退屈 → 行動 → 学習 → 理解

という循環が発生する。

実験環境
エージェントが接する世界は極めて単純である。

観測可能な対象：

温度
湿度
バッテリー
選択可能な行動：

急速冷却
通常冷却
加湿
現状維持
実験結果
成功した点
退屈（Novelty Hunger）が実際に行動動機として機能した。

学習によって世界理解が進むと退屈が増加し、現状維持を避ける傾向が確認された。

Voidは単なる観測値ではなく、行動生成機構として成立した。

判明した問題
世界が小さすぎる
エージェントが探索できる対象が温度・湿度・バッテリーしか存在しない。

そのため、

退屈 → 創造

ではなく、

退屈 → 介入

しか発生しなかった。

物理系が支配的
温度上昇の正帰還が強く、

温度上昇 → 熱漏洩増加 → 温度上昇

という発散構造が存在する。

後半はVoidの挙動よりも物理モデルの暴走が支配的になった。

有限性が存在しない
エージェントは事実上死なない。

バッテリーがマイナスになっても行動可能
極端な環境でも停止しない
そのため、

退屈と生存

の緊張関係が成立しなかった。

最も重要な発見
本実験で得られた最も重要な知見は、

「Voidは成立する」

ということではなく、

「Voidは世界を必要とする」

ということである。

欠損だけでは循環しない。

欠損は、

欠損 → 探索 → 世界 → 予想外 → 欠損更新

というループの中で初めて意味を持つ。

世界が十分に複雑でない場合、欠損は外界ではなく自己を掘り始める。

結論
HeuristicExplorerAI は恒常性維持AIではない。

また、一般的な報酬最大化エージェントでもない。

これは、

「理解すると退屈する知性」

を最小構成で実装した試みである。

実験の結果、退屈は確かに行動を生んだ。

しかし、温度・湿度・省エネだけの世界では、退屈は創造へ向かわず介入へ向かった。

本モデルはここで一旦完結とする。

次に必要なのは、より賢い行動アルゴリズムではなく、

より豊かな世界である。
